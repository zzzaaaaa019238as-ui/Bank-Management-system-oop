# 🏦 Bank Management System (OOP)

### 📝 Description
**Bank Management System** is a Python-based application that professionally implements **Object-Oriented Programming (OOP)** concepts like Encapsulation and Inheritance to manage banking operations.

---

### 🚀 Key Features
* 🏦 **Account Management**: Create and manage user accounts easily.
* 💸 **Transactions**: Secure deposit and withdrawal system.
* 🔒 **Data Security**: Using private attributes for sensitive data.
